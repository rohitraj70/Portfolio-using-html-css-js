# Frontend Developer Resume — Bold & Modern

A bold editorial-style resume website built with pure HTML, CSS, and vanilla JS. No frameworks, no build step, no dependencies.

## 🚀 Deploy to GitHub Pages (Free)

1. Create a GitHub repo named `yourusername.github.io`
2. Upload `index.html` + `README.md`
3. Go to **Settings → Pages → Source: main / root**
4. Live at `https://yourusername.github.io` in ~2 minutes

## ✏️ What to Update

Search and replace in `index.html`:

| Placeholder | Replace with |
|---|---|
| `YOUR NAME HERE` / `YN.` | Your actual name |
| `Your City, Country` | Your location |
| `Company Name`, `Agency Name` | Your real employers |
| `yourusername` | Your GitHub username |
| `yourprofile` | Your LinkedIn handle |
| `you@email.com` | Your email address |
| Project names & descriptions | Your real projects |
| Skills list | Your actual tech stack |

## Features

- Custom animated cursor (desktop)
- Tabbed experience section (no page reload)
- Scroll-reveal animations
- Responsive (collapses gracefully on mobile)
- Bebas Neue + DM Sans + DM Mono fonts
- Pure HTML/CSS/JS — zero dependencies
